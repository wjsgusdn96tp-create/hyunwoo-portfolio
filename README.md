# 포트폴리오 웹사이트

개인 포트폴리오 및 프로젝트 소개 웹사이트
https://wjsgusdn96tp-create.github.io/hyunwoo-portfolio/

## 기술 스택

- React
- React Router
- Vite
- Font Awesome
- react-big-calendar
- axios
- moment
- sweetalert2
- Chart.js

## 프로젝트 구조

- **메인 페이지**: 소개, 기술 스택, 프로젝트 목록
- **팀 프로젝트**: 세미 프로젝트, 파이널 프로젝트 상세
- **개인 프로젝트**: 일정 관리 캘린더

## 주요 기능

### 일정 관리 캘린더

- Google Calendar 스타일 UI
- 일정 CRUD 기능
- 시간 중복 체크
- 과거 날짜 등록 방지
- 한국어 지원

### 프로젝트 포트폴리오

- 카테고리별 프로젝트 필터링
- 반응형 디자인
- 상세 페이지 라우팅
